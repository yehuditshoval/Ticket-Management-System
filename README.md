# Ticket Management System

A professional full-stack Ticket Management System built with Angular (Frontend), C# (Backend), and SQL Database.

---

## Project Overview

This project demonstrates a complete ticket management system with:

- **Frontend:** Angular application with clean UI and dynamic components  
- **Backend:** C# application (ASP.NET or Console) handling business logic and API endpoints  
- **Database:** SQL scripts for creating tables, relationships, and sample data  

---

## Technologies Used

- **Frontend:** Angular, TypeScript, HTML5, CSS3  
- **Backend:** C#, .NET 6+, ASP.NET (optional)  
- **Database:** SQL Server / MySQL  
- **Version Control:** Git, GitHub  

---

## Features

- Ticket CRUD operations (Create, Read, Update, Delete)  
- Full integration between frontend, backend, and database  
- Database setup scripts included for easy replication  
- Structured for clarity and maintainability  

---

## Setup Instructions

1. **Database Setup**  
   - Navigate to `Database/` and run the provided SQL scripts to create tables and insert sample data.

2. **Backend Setup (C#)**  
   - Open the `Backend/` project in Visual Studio  
   - Restore NuGet packages  
   - Build and run the project to start API / console application

3. **Frontend Setup (Angular)**  
   - Navigate to `Frontend/`  
   - Run `npm install` to install dependencies  
   - Run `ng serve` to start the development server  
   - Open `http://localhost:4200` in your browser
